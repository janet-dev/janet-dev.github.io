Janet Dornan - Portfolio

Welcome to my portfolio; below is a collection of my past and current projects. To meet the needs of modern society, this site has been created to be mobile responsive.

This index (dot) html page is built using the Skeleton-2.0.4 - Responsive CSS Boilerplate. Skeleton is ideal for smaller projects or if all the utility of larger frameworks is not required. It only styles a handful of standard HTML elements and includes a grid, but that is often enough to get started.

Light and fast, at ~400 lines and built with mobile devices in mind. Styles are designed to be a starting point, not a UI framework. Quick to start with zero compiling or installing necessary.

For more information please read the Skeleton documentation
